🧮 Calculator Pro+

A modern, responsive, and feature-rich calculator built using HTML5, CSS3, and JavaScript. Designed with a clean interface, smooth animations, scientific functions, memory operations, calculation history, theme switching, and full mobile/desktop compatibility.

🚀 Features

🔢 Basic Calculations

- Addition
- Subtraction
- Multiplication
- Division
- Decimal Support
- Percentage Calculations

🧠 Scientific Functions

- Square Root (√)
- Square (x²)
- Inverse (1/x)

💾 Memory Functions

- MC (Memory Clear)
- MR (Memory Recall)
- M+ (Memory Add)
- M- (Memory Subtract)
- Memory Status Indicator

📜 Calculation History

- Automatically saves calculations
- History stored using Local Storage
- Clear History button

🌙 Theme Support

- Dark Mode
- Light Mode
- Theme preference saved automatically

⌨️ Keyboard Support

- Numeric Input
- Operators (+, -, ×, ÷)
- Enter Key Support
- Backspace Support
- Escape Key Support

🎨 Modern UI

- Glassmorphism Design
- Smooth Button Animations
- Full Width "=" Button
- Professional Color Palette
- Responsive Layout

💽 Local Storage

- Saves Theme Preferences
- Saves Calculation History
- Saves Memory Values

📱 Responsive Design

Fully optimized for:

- 📱 Smartphones
- 📲 Tablets
- 💻 Laptops
- 🖥️ Desktop Computers

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

🎯 Project Purpose

Calculator Pro+ was created to demonstrate modern front-end development techniques while providing a practical and visually appealing calculator experience.

This project showcases:

- DOM Manipulation
- Event Handling
- Responsive Web Design
- Local Storage Management
- UI/UX Design Principles
- Interactive JavaScript Features
